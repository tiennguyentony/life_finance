# Sprout Reference

`hero.png` is the canonical approved hero reference for Sprout. It defines the current body proportions, silhouette, face, leaf, feet, material, camera angle, lighting, and rendering treatment.

`palette.png` is generated from the canonical hero and provides a starting color reference for future asset work.

Do not overwrite it. Derive future assets from this reference and preserve the immutable rules in `/docs/CHARACTER-RULE.md`.
