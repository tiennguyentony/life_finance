# Sprout Reference

`sprout-main.png` is the canonical approved main character reference for Sprout. It defines the current body proportions, silhouette, face, leaf, feet, material, camera angle, lighting, and rendering treatment.

`sprout-money.png` is the approved money-gun variant.

`palette.png` is generated from the canonical main character and provides a starting color reference for future asset work.

Do not overwrite it. Derive future assets from this reference and preserve the immutable rules in `/docs/CHARACTER-RULE.md`.
